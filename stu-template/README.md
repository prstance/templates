# STU Template

Basic SolidJS + Typescript + UnoCSS template.


## Run Locally

Clone the project using [bun.sh](https://bun.sh)

```bash
  bunx degit prstance/templates/stu-template STU-template
```

Go to the project directory

```bash
  cd STU-template
```

Install dependencies

```bash
  bun install
```

Start the server

```bash
  bun run dev
```
